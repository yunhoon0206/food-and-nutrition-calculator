# 🍽️ 식품영양 분석 및 칼로리 계산 대시보드

## 📜 프로젝트 설명

본 프로젝트는 Streamlit을 활용하여 제작된 식품영양 정보 분석 및 칼로리 계산기 웹 애플리케이션입니다. 사용자들이 음식의 영양 정보를 쉽게 탐색하고, 자신의 식단을 관리할 수 있도록 돕는 것을 목표로 합니다.

---

## 🎯 주제

**음식 데이터 기반 영양 정보 탐색 및 맞춤형 칼로리 계산기**

---

## 🤔 주제 선택 이유

건강에 대한 관심이 높아지면서 식단 관리의 중요성이 커지고 있습니다. 하지만 식품의 영양 정보를 얻고 일일이 칼로리를 계산하는 것은 번거로운 일입니다. 이 프로젝트를 통해 사용자들에게 직관적인 데이터 시각화와 편리한 계산 기능을 제공하여 건강한 식습관 형성에 기여하고자 했습니다.

---

## 📊 데이터 분석 및 기능 소개

- **카테고리 별 음식 탐색**: 식품을 한식, 중식, 일식 등 다양한 카테고리로 나누어 탐색할 수 있습니다.
- **칼로리 Top 10**: 전체 음식 데이터 중 칼로리가 가장 높은 상위 10개 음식을 시각화하여 보여줍니다.
- **카테고리별 평균 칼로리**: 각 음식 카테고리별 평균 칼로리를 비교 분석하여 보여줍니다.
- **칼로리 계산기**: 사용자가 섭취한 음식들을 선택하면 총 섭취 칼로리를 자동으로 계산해줍니다.
- **음식 vs 음식 비교**: 두 가지 음식을 선택하여 영양 정보를 나란히 비교해볼 수 있습니다.
- **결제 페이지**: 서비스 구독이나 추가 기능을 위한 결제 시스템을 구현했습니다. (가상)
